# Image slots

Drop your own (owned or licensed) photos into this folder using the exact
filenames below. Each slot shows a warm placeholder until its file exists —
add a file and it appears automatically, no code changes needed.

Use your own or properly licensed images. Do not copy photos from another
company's website.

Team photos look best as square (1 : 1) head-and-shoulders shots.

Tip: keep files reasonably small (≈1600px wide max, optimised JPG/WebP)
for fast loading. `.jpg`, `.png` and `.webp` all work — just keep the
filename stem (e.g. rename a `.webp` to `hero.webp` and update the path,
or simply save as `hero.jpg`).
